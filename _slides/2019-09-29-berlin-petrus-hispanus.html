AIMS OF THE SCTA HTTP://SCTA.INFO The SCTA exists to be a new type of social community dedicated to the publication of scholastic texts divorced from presentation, freeing that data to be used in a plurality of presentations for a plurality of use cases.
WHAT THE SCTA DOES Maintains a set of domain specific standards for the publication of textual data as data anywhere on the web. Helps publish this decentralized data by organizing it with detailed metadata. Makes this metadata available through various APIs that client applications can make use of to display this data in any way desired.
Why we need something like the SCTA Sustainable Open Access without Redundancy Enabling Connected Connectivity and Discoverabilty
